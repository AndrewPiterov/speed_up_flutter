# SpeedUp Flutter

This package will speed up your developing.

## Usage

### Number extensions

```dart
20.h; // Add vertical spacing
20.w; // Add horizontal spacing
```

### Section

A section is a thematic grouping of content, typically with a heading

```dart
Section(
  [
    Text('Name Andrew'),
    10.h,
    Text('Age 18'),
  ],
  heading: Text('User info'),
)
```
